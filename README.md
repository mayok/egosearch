# これなん
twitter えごさ bot

# インストール方法
以下のコマンドを実行する。

```
$ cd /path/to/hogehoge
$ git clone git://github.com/mayok/egosearch.git
$ bundle install --path vendor/bundle
```

# 使い方
config.yml の中身を書いてから実行してください。

```
$ cp config.yml.sample config.yml
$ [editor] config.yml
$ bundle exec ruby search.rb
```
